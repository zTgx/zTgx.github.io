工作中, 要编译一个C++库, 要求cmake版本 > 3.5, 而目前版本是2.x, 需要更新.  
下载最新版的cmake源码, 执行以下命令
```c
./cofigure && sudo make && sudo make install
```
然后 查看cmake版本
```c
cmake --version
```
于是有了题目的报错信息, 原因类似于缓存导致的失败.

解决方法:
```c
hash -r
```
