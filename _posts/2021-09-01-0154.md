## 唯一原则
* 可读性是唯一原则。代码是为人阅读的，不能使用奇技婬巧而损失了可读性。

## 内容更新

### 9.1 - 更新
* preclude 内容就用 `*`声明，不用展开，如
```rust
use ruc::*;
```

* 文档注释和源码空格一行，如 `a.rs`
```rust
//!
//! data sources for the query api
//!

use bnc::map;
```
